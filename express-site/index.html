Setting up an Express server

In the terminal, run git init and npm init -y.

Create a file named .gitignore with the following lines:

/node_modules
/.vscode
/DS_Store
.env

Create a folder in the root directory containing an index.html file.

Create the file src/server.js containing the following code. (Replace "mywebsite" with the name of the folder you made earlier)

const express = require("express");

const port = 5001;
const app = express();

app.use("/mywebsite", express.static("mywebsite"));

app.listen(port, () => {
    console.log(`Server is listening on port ${port}`);
});
